IntelliCart — Real-Time Grocery Detection & Automated Billing

Amazon Go works — but it costs millions in hardware per store, locking out smaller retailers. IntelliCart closes that gap: a camera-only checkout system using YOLOv8s that detects grocery items in real time and generates an itemised bill automatically. No barcode scanner. No specialised hardware. Just a webcam and a trained model. Benchmarked against SSD-MobileNet (~46 FPS, mAP 0.2832) and Faster R-CNN (mAP 0.8932). Built on FastAPI + Next.js, deployed on Railway + Vercel.
